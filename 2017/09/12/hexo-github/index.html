HEXO搭建个人博客

现大四生，想走web前端方向，平时学文章不多，为了有个良好习惯，搭建个人博客网站，存放笔记、趣事文章，朋友介绍用hexo，简单快速，尝试一下，为自己积累经验.

###Hexo
Hexo(中文官方网站)是一个快速，简洁且高效的博客框架. 让上百个页面在几秒内瞬间完成渲染. Hexo支持Github Flavored Markdown的所有功能， 甚至可以整合Octopress的大多数插件. 并自己也拥有强大的插件系统.

#1.安装Github和NodeJs

  1. Github下载地址

    命令行查看Github版本: git version.

  2. NodeJs下载地址

    命令行查看NodeJs版本: node -v.

注：1.安装NodeJs时，在Custom Setup中选择add to path. 2.打开下载好的Git里的git-bash.exe书写命令.

#2.安装Hexo

  1. 在任意盘中建立一个文件夹来存放hexo，如D:\mybolg.
  2. 命令行cd到mybolg文件夹下.

  3. 安装hexo命令：$ npm install -g hexo-cli

    注：一直未响应，可用国内npm镜像安装
    $ npm install -g cnpm --registry=https://registry.npm.taobao.org

  4. 初始化命令：hexo init

     node_modules：是依赖包
 public：存放的是生成的页面
 scaffolds：命令生成文章等的模板
 source：用命令创建的各种文章
 themes：主题
 _config.yml：整个博客的配置
 db.json：source解析所得到的
 package.json：项目所需模块项目的配置信息

#3.Hexo本地配置

  1. 打开myblog文件，修改_config.yml文件的一些配置

     `deploy:
   type: git
   repo: https://github.com/YourgithubName/YourgithubName.github.io.git
   branch: master`

    注：每个冒号后都有一个空格,YourgithubName为你的Github账户名.

  2. 安装：npm i hexo-server

  3. 命令行：

     `hexo clean 
  hexo generate
  hexo server`

  4. 打开浏览器输入：http://localhost:4000，本地上传成功了
    #4. Github账户的注册和配置

  5. Github注册,官网网址

    注：注册时填写邮箱，点击Github发送注册确认信，确认注册，结束注册流程.

  6. 创建仓库
    登陆之后，点击页面右上角的加号，选择New repository
    在Repository name下填写yourname.github.io，Description (optional)下填写一些简单的描述（可不写）.

  7. 配置Github账户信息
    返回git.bash.exe中，输入：

    git config --global user.name "YourName"

    git config --global user.email "YourEmail"

    注：YourName和YourEmail是注册Github的账户名和邮箱.

  8. 创建SSH，命令行输入：ssh-keygen -t rsa -C "youremail@example.com

    注：ssh生成失败的情况下，需要在电脑-属性-高级系统属性-环境变量中将系统变量中的path，添加值：ssh-keygen.exe的路径，路径在Git/usr/bin目录下，找不到可以搜索下.

  9. 生成ssh后，会有两个文件，id_rsa.pub和id_rsa将id_rsa.pub打开，拷贝内容放到Github中.Github官网打开Settings中SSH and GPG keys，创建新的ssh,NEW SSH KEY.



#5.联系Hexo与Github page

  1. 安装：npm install hexo-deployer-git --save,才能将博客部署到到github服务器上并可供人浏览.

  2. 每次部署必备步骤

      hexo clean
   hexo generate
   hexo deploy

    注：在hexo deploy过程中要输入你的Github账户名和密码.
    在浏览器中输入http://yourgithubname.github.io,就打开你的个人微博了.
